# Project Memory

## Core
Multi-page architecture with proper inter-page routing (no single-page scroll).
Retain base template UI framework while applying new specific redesign content/style rules.
Font family: Montserrat for both display and body text everywhere.
Navigation items strictly limited to: Solutions, About, Career, Contact (no dropdowns).
Images must be strictly square. NO rounded corners, NO decorative shapes or diagonal frames.
Aesthetic: Enterprise-grade, minimal, "invisible yet powerful". Calm motion, subtle data flows.
Investor Calendly: https://calendly.com/minetech-rwanda/30min
SEO: every page uses `<SEO>` from `@/components/SEO`. JobDetail emits JobPosting schema.
